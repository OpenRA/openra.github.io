http://www.openra.net/ OpenRA - News Feed 2015-04-24T16:00:00Z OpenRA developers http://www.openra.net tag:www.openra.net,2015-04-24:/news/release-20150424/ Release 20150424 2015-04-24T16:00:00Z 2015-04-24T16:00:00Z <p>We are very pleased to announce the Spring release of OpenRA!</p> <p>This release a huge step forward for the Dune 2000 mod in particular. It includes just shy of 1000 commits by 53 contributors, the culmination of six months of relentless work. With more than 2000 files and 210,000 changed lines it is one of our busiest development cycles to date.</p> <p>A large development focus has been on the Dune 2000 mod, which contains the most visible changes. A new sidebar, sandworms and Carryalls are the big new features, which should make any Dune 2000 player feel more at home in our mod. Additionally, the OpenRA.Utility now supports importing maps from the original Dune 2000 game, so we hope that the number of maps and of course the number of players will increase in the near future!</p> <p>The other mods were not neglected, though. In Red Alert, the standout feature is the support of playable factions such as Russia, England, France, Ukraine and Germany. In both Red Alert and Tiberian Dawn, new single-player missions are available that will keep you occupied.</p> <p>Of course, we have not only improved the mods themselves. Here is a list of changes that affect the game in general:</p> <ul> <li>The server browser shows and allows joining games from any installed mod.</li> <li>It is now possible to disable the “short game” mode that was forced in the past.</li> <li>The player name and colour can now be set up in the settings dialog.</li> <li>When creating a server, the game informs the host about port forwarding issues.</li> <li>Left-click mouse orders have been markedly improved.</li> <li>Some missions now support difficulty levels.</li> <li>A huge number of smaller gameplay fixes and improvements, including: <ul> <li>Friendly units blocking the exit of a production building will now move out of the way if needed.</li> <li>Units will no longer hunt cloaked units.</li> <li>The spy plane in Red Alert will no longer lure your anti-air capable units across the map.</li> <li>Planes will no longer collide with helicopters and push them out of the way.</li> <li>Your faction will not be revealed to enemies when you have selected a random faction in the lobby.</li> <li>In sell mode, the tooltip will now show the expected refund value for structures.</li> <li>Harvesters should now no longer block themselves at refineries, and also make use of all available refineries if the nearest ones are occupied.</li> </ul> </li> <li>A similarly massive number of performance improvements have been made in this release.</li> <li>Many race conditions and crashes have been fixed across the board.</li> </ul> <p>For the complete list of changes, please visit <a href="http://changelog.openra.net">our changelog page</a>.</p> <p>A note on key bindings: In the previous release, the key to toggle team chat was changed to ‘Alt’, which, looking back, was an unfortunate decision. It has been reverted to ‘Tab’ in this release, where it serves a double role: on an empty line, it toggles team chat, otherwise it auto-completes player names.</p> <p>For modders and mappers, there have been quite a few important changes as well. We have introduced a new map format, to which maps running on this release will be updated automatically. All of the shared engine code has been moved into the OpenRA.Mods.Common project, so that third-party mods can drop their dependencies on our game mods. A lot of code has been changed to enable features that will be needed by the Tiberian Sun mod. One important change for Red Alert mappers is that the gem mine has been replaced with a new one. For more details, check out the <a href="http://changelog.openra.net">changelog</a>, as well as the <a href="https://github.com/openra/openra/wiki/Traits">trait</a> and <a href="https://github.com/openra/openra/wiki/Lua-API">Lua API</a> documentation.</p> <p>Looking forward, we hope to get the next release out faster than this time around (famous last words!). Feature-wise, you can probably expect the first appearance of the in-game map editor in the next release, in addition to much of the things that are currently in <a href="https://github.com/openra/openra/pulls">our pull request queue</a>. Beyond that, we will see… If you want to keep track of the development, keep an eye on the <a href="https://github.com/OpenRA/OpenRA/milestones/Next%20release">release milestone</a> and the <a href="https://github.com/openra/openra/wiki/Changelog%20(bleed)">development changelog</a>.</p> <p>We will also try to make more use of <a href="https://www.youtube.com/channel/UCRoiPL1J4K1-EhQeNazrYig">our YouTube channel</a> to show off new and little known existing features, provide mission walkthroughs and other content, so watch and subscribe!</p> <p>Now enough with the talking! Head on over to our <a href="/download/">download section</a>, grab the installer for your OS, and enjoy the game!</p> <div style="text-align:center"> <iframe width="600" height="333" src="https://www.youtube-nocookie.com/embed/oK-paA7kRP0?rel=0" frameborder="0" allowfullscreen=""></iframe> <p>The long awaited sandworms and Carryalls have finally arrived in our Dune 2000 mod!</p> <p><img src="/images/news/20150424-ra-country-dropdown.png" alt="Playable countries" /></p> <p>Red Alert now has playable country factions, and matching choices for random faction selection.</p> <p><img src="/images/news/20150424-cnc-server-browser.png" alt="Overhauled server browser" /></p> <p>The server browser now lists games from all available mods, and also allows joining games across mods.</p> <p><img src="/images/news/20150424-cnc-mission-difficulty.png" alt="Mission difficulty" /></p> <p>Some missions have received support for multiple difficulty levels.</p> <p><img src="/images/news/20150424-ra-refund-tooltip.png" alt="Refund tooltip" /></p> <p>In sell mode, the tooltip will now show a structure’s expected refund value.</p> <p><img src="images/news/20150424-ts-teaser.png" alt="Tiberian Sun teaser" /></p> <p>Behind the scenes, the still unreleased, work-in-progress Tiberian Sun mod has received support<br />for some crucial features.
